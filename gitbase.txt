git clone 주소 : 원격 저장소에서 로컬 저장소로 불러오기

git push 별칭 브랜치 : 로컬 저장소 내용을 원격 저장소로 보내기
git pull 별칭 브랜치 : 원격 저장소 내용을 로컬 저장소로 불러오기
